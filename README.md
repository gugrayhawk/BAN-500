# BAN-500

The files in this repository are as follows

## BAN 500 Hurricanes Data.xlsx

Using statistics from the magical 2018-19 season, create an interactive Excel spreadsheet to allow analysis using W/L, Shots taken, and opponents. 

## BAN 500 – Spotify Music.xlsx

Using data from Spotify, analyze various attributes of music for danceability, energy, and popularity. 

## Pinball Dashboard.xlsx

The final project for this class. Using data about the top 300 pinball machines from 1975-2014, create an interactive spreadsheet to see popularity by manufacturer, Licensing, and 5 year intervals.
